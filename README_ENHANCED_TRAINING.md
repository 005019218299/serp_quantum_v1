## 🔍 Troubleshooting

### Lỗi thường gặp:

1. **"All strategies failed"**
   - Kiểm tra internet connection
   - Thử lại sau 5-10 phút
   - Giảm số keywords

2. **"Insufficient data"**
   - Tăng số keywords (>5)
   - Kiểm tra keyword quality
   - Thử keywords phổ biến hơn

3. **"Model training failed"**
   - Kiểm tra RAM (cần >2GB)
   - Restart Python session
   - Xóa cache cũ
